这是一篇示例博客文章，支持 **Markdown** 与 GFM 表格、任务列表等（由 `remark-gfm` 提供）。

## 你可以怎么改

1. 编辑 `src/data/profile.ts` 填写姓名、职位、邮箱与外链。
2. 编辑 `src/data/projects.ts` 替换为你的真实项目与 GitHub 链接。
3. 在 `src/content/blog/` 下复制本文件，修改 `title`、`date`、`slug`（文件名即 slug）与正文。

## 小表格示例

| 模块   | 路径                 |
| ------ | -------------------- |
| 项目数据 | `src/data/projects.ts` |
| 个人简介 | `src/data/profile.ts`  |
| 文章   | `src/content/blog/*.md` |
